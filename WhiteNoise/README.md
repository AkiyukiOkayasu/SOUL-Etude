# WhiteNoise  
White noise generator using SOUL helper class [**soul::noise**](https://github.com/soul-lang/SOUL/blob/master/source/modules/soul_core/library/soul_library_noise.h)

## Compile  
```bash
soul play WhiteNoise.soulpatch
```
